[GET]: List Tenants#

This method can be used to retrieve a list of current tenants

Note: This method restricts tenant database and user information when the x-api-key token holder is a restricted super admin Note: This method shows only tenants that have admin ui users when the x-api-key token holder is a super_admin_ui

Input#

GET /admin/tenants

Request headers#

x-api-key: shared_key_value
Accept: application/json

Response#
